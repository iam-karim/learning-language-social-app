# 🌍 Language Learning Platform

> A full-stack language learning platform built to gain hands-on experience with authentication, REST APIs, real-time messaging, and video communication using modern web technologies.

> 🚧 **Status:** Learning Project • Actively Being Extended

---

## 📖 About

This project was initially built while following a full-stack learning course to understand how modern web applications are designed and implemented.

Rather than treating it as a tutorial exercise, I use this repository as a foundation for experimenting with new backend concepts, improving the architecture, and implementing additional production-oriented features.

The long-term goal is to evolve this project into a complete language learning platform with unique functionality beyond the original implementation.

---

## ✨ Current Features

- Secure User Authentication
- JWT Authentication
- HTTP-only Cookie Authentication
- User Profiles
- Real-Time Messaging
- Video Calling
- React Query Data Fetching
- Zustand State Management
- Responsive UI
- Theme Switching
